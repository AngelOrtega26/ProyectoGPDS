# ***BACKEND***

### API de NodeJS con arquitectura en capas

#### *Modulos usados en esta API:*
- express
```
npm install express
```
- postgres
```
npm install pg
```
- dotenv (para hacer uso de variables de entorno y no exponer el URI directamente el el API)
```
npm install dotenv
```
- nodemon
```
npm install --save-dev nodemon
```
